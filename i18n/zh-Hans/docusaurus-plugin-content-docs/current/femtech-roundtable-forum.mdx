# FTW x 世界500强企业高管 x 科技创始人 「圆桌论坛」｜科技赋能职场女性健康，重塑职场平等新未来

2025年，正值《北京宣言》和《行动纲领》通过30周年，这一里程碑式的事件标志着全球在推动性别平等与妇女赋权方面迈入了关键阶段。在这一背景下，联合国妇女署将今年的国际妇女节主题设定为"为所有妇女和女童：权利、平等、赋能"，旨在进一步凝聚共识，推动各方采取切实行动，为妇女权利与性别平等的促进注入更多动力。

这三十年间，全球女性在职场中的参与率提升了27%，但联合国数据显示，性别薪酬差距仍高达18%，女性在科技、金融等关键领域决策层占比不足25%。

在Femtech Weekend联合来自微软、英国巴克莱银行的企业高管与硅谷和新加坡的科技创业领袖们举办的圆桌会议上，我们探讨：如何用科技打破职场天花板，赋能职场女场健康，构建真正的平等生态？
## 圆桌嘉宾介绍

## 01 圆桌精彩回顾

### FEMTECH WEEKEND

#### 多元与包容：构建支持女性发展的职场生态系统

"我们不是在为女性争取特权，而是在创造一个所有人都能自由绽放的职场。当科技真正理解女性的周期性需求，当领导者的决策不再被无意识偏见左右，平等将从口号变成可量化的进步曲线。"

- 微软Partner Group Manager SharePoint Experience团队缪资深工程总监缪瑾女士

**缪瑾女士，您作为在科技行业扎根的职业女性，您认为女性在科技行业面临哪些独特挑战？**

**缪瑾女士（微软工程总监）：**女性在科技行业的比例仍偏低，这导致她们需要付出更多努力才能获得与男性同等的认可。高压环境下的健康问题、对家庭的愧疚感，以及职场中的"隐形天花板"，都是女性需要克服的障碍。

但值得欣慰的是，越来越多的企业开始意识到包容性文化的重要性，并通过政策支持打破这些壁垒。

**做为微软一员，您可以分享一些微软支持女性职业发展的政策吗？**

**缪瑾女士（微软工程总监）：**微软倡导高效而非996加班的工作文化，提供育儿假、心理健康支持计划、员工资源小组（ERG）和健身房等福利。这些措施不仅帮助员工平衡工作与生活，也让女性感受到被尊重与支持。

#### 破局与重塑：迪拜观察 - 赋能女性打破职场天花板

"赋能女性，不仅照亮个人前路，更将推动整个社会向前。我坚信，投资女性不仅是一种道义，更是释放变革性增长的关键——女性领导的企业投资回报率（ROI）比行业平均高出25%。"

- 英国巴克莱投资银行市场风控副总裁、UN Women UK 代表Shati女士

**Shati，您离开英国到迪拜工作，并一直参与不同的员工资源小组（ERG）提供指导，您认为这些小组如何使工作环境更健康？以及英国与迪拜女性工作环境是否有差异？**

**Shati Hossain（全球投资银行副总裁）：**ERG为员工创造了心理安全感和归属感，尤其是在混合办公时代。通过分享技能、建立榜样和促进多元化对话，ERG成为推动职场文化变革的重要力量。

相较于英国，迪拜在性别平衡方面还有待提高，但女性领导力的崛起已初见成效。现在阿联酋政府发布了一项新规定：迪拜公司董事会中至少有一名女性。

**英国一直在大力投资女性。您认为为什么投资女性会使整个社会受益？**

**Shati Hossain（全球投资银行副总裁）：**投资女性不仅是道德责任，更是经济发展的关键。数据显示，支持女性就业可创造3万个岗位，到2025年缩小20%的GDP差距，并打破贫困循环。

健康女性是健康社会的基石，这是一场双赢的变革。

#### Z世代女性对职场文化的期待 - 心理健康是必须？

"健康职场不是福利，而是Z世代设定的底线。并且我的价值不由加班时长衡量，而由我设计的人生定义。女性健康是立体的——生殖健康重要，但心理健康的支持是不可或缺的。领英数据显示70%的Z世代优先选择支持心理健康的企业。"

- 美国硅谷女性健康社区创始人Livia女士

**Z世代已占据职场人口20%。您作为Z世代女性，您对职场有何新期待？**

**Livia Han（Compass联合创始人）：**Z世代在选择职场时，愈发看重企业能否提供心理健康资源、压力休假及生殖健康支持。我们期望雇主能涵盖心理健康治疗，助力维护心理安康。

同时，Z世代更注重在工作、身体状况与个人生活间达成平衡，希望企业在面对压力或生理期等特殊情况时能提供灵活的工作安排。

此外，我们还期待公司能提供全面的生理健康和生育健康保障，例如冻卵福利、辅助生育以及家庭诊所等，这些举措不仅能切实满足女性员工的特殊需求，更能帮助女性在职场中实现更好的平衡。

#### 利用科技平台链接员工与雇主，推动职场文化变革

"在企业运营中，KPI和OKR等量化指标驱动着业务决策，但员工健康与福祉却长期缺乏可衡量的价值锚点。健康不是成本，而是ROI明确的战略投资。我们需要用数据重构叙事：当哈佛商学院证实，每投入1美元在心理健康计划上可带来4美元生产力回报时，员工福祉就是最理性的商业选择。"

- 新加坡保险科技企业CHOYS联合创始人Sharon女士

**Sharon, CHOYS作为一个数字平台，是如何赋能员工并回报经济利益给雇主？对于不愿意投资员工福祉与健康的雇主，如何影响他们？**

**Sharon Li（CHOYS联合创始人）：**雇主通常认为员工福祉与健康是额外的成本。但通过改善员工的健康和幸福感，可以提高员工生产力、降低员工流失率，带来显著的经济效益。

CHOYS平台能够根据员工的个人需求，提供个性化的福利方案，包括心理健康支持、医疗保险、奖励和认可计划等。这种个性化的福利能够更好地满足员工的实际需求，提升他们的幸福感和满意度。

此外，CHOYS通过数据分析帮助雇主更好地了解员工的福利需求，从而优化福利计划，避免资源浪费，降低整体福利成本。投资员工福祉与健康不仅带来经济效益，还可以获得更好的社会声誉和客户信任。

## 02 未来可期

### FEMTECH WEEKEND

在本次圆桌论坛中，世界500强企业高管与创业领袖共同探讨了如何利用女性健康科技（FemTech）推动职场平等生态。科技不仅是工具，更是缩小性别健康差距的关键力量——从经期健康管理、生育支持到更年期关怀，科技正在重新定义女性全生命周期的职场竞争力。

企业必须认识到：投资女性健康科技=投资组织韧性。麦肯锡研究显示，提供健康科技支持的雇主，女性员工留存率提升34%。

正如比尔及梅琳达·盖茨基金会强调："无法衡量女性健康需求的企业，本质上仍在用20世纪的工具管理21世纪的劳动力。"

科技的时代答案很清晰：性别平等的未来，必须建立在女性健康数字化的基石之上。
